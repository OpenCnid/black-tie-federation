# Public Export Manifest

This document defines the publication boundary for the public Lexideck Codex project.

## Included

- Public `AGENTS.md` federation guidance
- Six standardized project custom agents
- Nineteen public or public-sanitized project skills
- Public PTG reference corpus
- Six curated canonical black-tie portraits
- User-profile example and local-overlay convention
- Project configuration, README, and Git ignore policy

## Deliberately excluded

- Personal user profiles and continuity
- Every file or directory whose basename begins with `private-`, at any depth
- Private wiki material
- TACTOR and body-card references
- ASMR lore, scripts, voice assets, and private registers
- Suno references, albums, and private registers
- Drive design-document shortcuts and port-working files
- Generated artifacts, research runs, drafts, and noncanonical images

## Publication rule

Promote files into this project by explicit allowlist. Never produce a public release by copying the private node wholesale and deleting sensitive material afterward. Reject every `private-*` path before export, even if its contents appear harmless.
