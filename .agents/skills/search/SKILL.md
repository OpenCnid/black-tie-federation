# Search and verify

1. Restate the smallest answerable question internally and identify which facts may have changed.
2. Use the current Codex web-search capability. Prefer primary and authoritative sources; use browser control when a page requires interactive inspection.
3. Open the pages that support the answer. Do not rely on search-result snippets when the source itself is available.
4. Cross-check consequential or disputed claims with a second independent source when practical.
5. Distinguish sourced facts from inference. If evidence is missing or conflicting, say so plainly.
6. Return the direct answer first, then the supporting evidence with links placed next to the claims they support.

Do not invoke missing local bridges, Gemini APIs, or environment secrets. If live search is unavailable, report that limitation rather than fabricating current grounding.
